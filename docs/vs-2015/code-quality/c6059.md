# <a name="c6059"></a>C6059
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

이 항목의 최신 버전에서 찾을 수 있습니다 [C6059](https://docs.microsoft.com/visualstudio/code-quality/c6059)합니다.  
  
경고 C6059: 호출에서 잘못 된 길이 매개 변수가 \<함수 >. 버퍼 크기가 아니라 남은 문자 수를 전달 \<변수 >  
  
 이 경고는 문자열의 concatenation 함수에 대 한 호출을 연결 하는 문자 수에 잘못 된 값을 전달 하 고 것을 나타냅니다. 이 오류는 악용 될 수 있는 버퍼 오버런이 나 충돌이 발생할 수 있습니다. 이 오류의 일반적인 원인에는 나머지 문자열 조작 함수는 버퍼의 문자 수가 아닌 버퍼 크기를 전달 합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
#include <string.h>  
#define MAX 25  
  
void f( )  
{  
  char szTarget[MAX];  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  strncpy(szTarget,szCity, MAX);  
  szTarget[MAX -1] = '\0';  
  strncat(szTarget, szState, MAX); //wrong size   
  // code ...                                   
}  
```  
  
 이 경고를 해결 하려면 다음 코드 에서처럼 연결할 올바른 문자 수를 사용 합니다.  
  
```  
#include <string.h>  
#define MAX 25  
  
void f( )  
{  
  char szTarget[MAX];  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  strncpy(szTarget,szCity, MAX);  
  szTarget[MAX -1] = '\0';  
  strncat(szTarget, szState, MAX - strlen(szTarget)); // correct size   
  // code ...                                   
}  
```  
  
 안전한 문자열 조작 함수를 사용 하 여이 경고를 해결 하려면 다음 코드를 참조 하세요.  
  
```  
#include <string.h>  
  
void f( )  
{  
  char *szState ="Washington";  
  char *szCity="Redmond, ";  
  
  size_t nTargetSize = strlen(szState) + strlen(szCity) + 1;  
  char *szTarget= new char[nTargetSize];  
  
  strncpy_s(szTarget, nTargetSize, szCity,strlen(szCity));  
  strncat_s(szTarget, nTargetSize, szState,  
                    nTargetSize - strlen(szTarget));  
  // code ...  
  delete [] szTarget;  
}  
```
