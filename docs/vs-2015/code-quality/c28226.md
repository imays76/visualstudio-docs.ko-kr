# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28226: 또한 주석은 현재 선언에서 PrimOp일 수 없습니다.  
  
 이 경고는 명명된 주석을 PrimOp로 선언하는 중이며, 이전에 일반적인 주석으로 선언되었음을 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
