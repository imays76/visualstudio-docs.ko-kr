# <a name="modules"></a>모듈
디버거 아키텍처 측면을 *모듈*:  
  
-   코드 예: 실행 파일 또는 DLL의 물리적 컨테이너입니다.  
  
-   해당 기호를 다시 로드 하 고 스스로 설명할 수 있습니다. 모듈 설명 IDE의 모듈 창에 표시 됩니다.  
  
-   로 표시 됩니다는 [IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md) 모듈을 설명 하는 디버그 엔진에 의해 생성 된 인터페이스입니다.
