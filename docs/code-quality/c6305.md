# <a name="c6305"></a>C6305
경고 C6305: sizeof와 countof의 개수가 잠재적으로 일치 하지

 이 경고 변수 보유 한 `sizeof` 결과 더하거나 뺄 포인터에서 또는 `countof` 식. 그러면 포인터 산술 연산에서 예기치 않은 크기를 조정 합니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성합니다.

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 이 경고를 해결 하려면 다음 코드를 사용 합니다.

```
void f(int *p)
{
  // code...
  p += 1;
}
```
